# Автомат сортировки массива + PS_2
Реализован конечный автомат, который принимает массив чисел по PS_2 протоколу, и сортирует по возрастанию merge sort'ом

# Модули
 - clk_divider.v - делитель частоты
 - count.v - счетчик
 - filtercon.v - филтьр дребезга
 - count.v - счетчик
 - fsm.v - Конечный автомат
 - main.v - модуль верхнего уровня
 - PS2_DC.v - дешифратор PS_2
 - PS2.v - модуль считывания PS_2
 - PS2_manager - общий модуль PS_2
 - SevenSegmentLED.v - модуль управления семисегментным индекатором
 - synchro.v - синхронизатор
