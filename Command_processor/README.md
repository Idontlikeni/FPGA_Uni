# CORDIC + ШИМ
Данный проект является простым обработчиком команд, способный выполнять суммирование, запись данных в память, и вывод данных на светодиоды.
Ввод и вывод данных проводится с помощью протокола UART

## Модули:
 - TT_UART.v - Модуль протокола UART
 - TT_TOP.v - модуль верхнего уровня
 - TT_ANALYZER.v - анализатор последовательностей
 - TT_CNTRL_LEDS.v - модуль управления светодиодами
 - TT_DC_ASCII_HEX.v и TT_DC_HEX_ASCII.v - дешифратор ASCII -> HEX и HEX -> ASCII
 - TT_DIVIDER.v - делитель частоты
 - TT_GEN_MSG.v - генератор сообщений (на вывод юарта)
 - TT_HANDLER_CMD.v - обработчик команд
 - TT_INFS_8B.v - 
 - TT_TOP.v - модуль верхнего уровня.
 - counter.v - счетчик
