# Репозиторий с FPGA проектам
В этом репозитории находятся source-файлы моих проектов. В каждой папке находятся как и модули, так и .mem файлы, так и файлы симуляции.

В каждой папке находится свой README.md с описанием модулей.
 - Command_processor - Обработчик команд с протоколом UART.
 - CORDIC - Реализация CORDIC алгоритма (радуга на светодиодах)
 - Pipeline_CPU - Простой конвеерный процессор
 - PS2_Sort - Конечный автомат сортировки слиянием с вводом данных по PS2 протоколу.
 - UART - Протокол UART (То же самое что и в Command_processor)

Все проекты реализованы на базе ПЛИС digilent artyx A7.
