# CORDIC + ШИМ
Данный проект реализует плавный переход-радугу на 3х светодиодах используя CORDIC и ШИМ.
С помощью двух пар кнопока пользователь может управлять скоростью смены цвета и яркостью светодиодов.

## Модули:
 - debouncer.v - фильтр дребезга
 - clk_divider.v - делитель частоты.
 - counter.v - счетчик
 - SevenSegmentLED.v - модуль управления семисегментным индикатором.
 - PWM_FSM.v - Автомат управления яркостью/частотой
 - main.v - Верхний модуль
